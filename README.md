# pacha-codex-dev
Pacha Codex is a living knowledge ecosystem mapping the history, memory, and culture of the Americas through interconnected nodes, dynamic semantic projections, and multiple simultaneous ontologies. This repository is the development laboratory — where the system is built, broken, and rebuilt from the ground up.
